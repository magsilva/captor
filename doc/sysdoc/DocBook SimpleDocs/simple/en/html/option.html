option — An option for a software command

Synopsis

Mixed Content Model

option ::=
(#PCDATA|replaceable|inlinemediaobject)*

Attributes

Common attributes

Description

Option identifies an optional argument to a software command.

Processing expectations

Formatted inline.

DocBook does not specify whether or not a symbol (such as - or /) is generated before the content of Option, or what that symbol might be. Generating the text may or may not be desirable, but in either case, it is an interchange issue.

Future Changes

The content model of Option will be constrained to (#PCDATA | Replaceable | InlineGraphic) in DocBook V4.0.

Parents

These elements contain option: attribution, bibliomisc, citetitle, command, computeroutput, emphasis, entry, lineannotation, link, literal, literallayout, para, phrase, programlisting, quote, subtitle, term, title, titleabbrev, trademark, ulink, userinput.

Children

The following elements occur in option: inlinemediaobject, replaceable.

See Also

computeroutput, literal, replaceable, userinput.
