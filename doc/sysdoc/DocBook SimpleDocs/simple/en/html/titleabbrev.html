titleabbrev — The abbreviation of a Title

Synopsis

Mixed Content Model

titleabbrev ::=
(#PCDATA|footnoteref|xref|abbrev|acronym|citetitle|emphasis|
 footnote|phrase|quote|trademark|link|ulink|command|
 computeroutput|email|filename|literal|option|replaceable|
 systemitem|userinput|author|corpauthor|othercredit|revhistory|
 inlinemediaobject)*

Attributes

Common attributes

Description

TitleAbbrev holds an abbreviated version of a Title. One common use of TitleAbbrev is for the text used in running headers or footers, when the proper title is too long to be used conveniently.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Abbreviated titles are usually used only in specific contexts, such as headers and footers, and suppressed everywhere else.

Parents

These elements contain titleabbrev: appendix, article, articleinfo, bibliodiv, bibliography, bibliomixed, bibliomset, example, figure, itemizedlist, objectinfo, orderedlist, section, sectioninfo, sidebar, variablelist.

Children

The following elements occur in titleabbrev: abbrev, acronym, author, citetitle, command, computeroutput, corpauthor, email, emphasis, filename, footnote, footnoteref, inlinemediaobject, link, literal, option, othercredit, phrase, quote, replaceable, revhistory, systemitem, trademark, ulink, userinput, xref.
