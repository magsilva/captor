email — An email address

Synopsis

Mixed Content Model

email ::=
(#PCDATA|link|ulink|emphasis|trademark|replaceable|
 inlinemediaobject)*

Attributes

Common attributes

Description

Inline markup identifying an email address.

Processing expectations

Formatted inline. Email may generate surrounding punctuation, such as angle brackets. This is an interchange issue.

In some processing environments, Email may automatically generate a hypertext link (a mailto: URL).

In an Address, this element may inherit the verbatim qualities of an address.

Parents

These elements contain email: attribution, bibliomisc, citetitle, command, computeroutput, emphasis, entry, lineannotation, link, literal, literallayout, para, phrase, programlisting, quote, subtitle, term, title, titleabbrev, trademark, ulink, userinput.

Children

The following elements occur in email: emphasis, inlinemediaobject, link, replaceable, trademark, ulink.
