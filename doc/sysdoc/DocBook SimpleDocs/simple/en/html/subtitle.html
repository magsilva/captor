subtitle — The subtitle of a document

Synopsis

Mixed Content Model

subtitle ::=
(#PCDATA|footnoteref|xref|abbrev|acronym|citetitle|emphasis|
 footnote|phrase|quote|trademark|link|ulink|command|

 systemitem|userinput|author|corpauthor|othercredit|revhistory|
 inlinemediaobject)*

Attributes

Common attributes

Description

A Subtitle identifies the subtitle of a document, or portion of a document.

Processing expectations

Formatted as a displayed block.

Parents

These elements contain subtitle: appendix, article, articleinfo, bibliodiv, bibliography, bibliomixed, bibliomset, objectinfo, section, sectioninfo.

Children

The following elements occur in subtitle: abbrev, acronym, author, citetitle, command, computeroutput, corpauthor, email, emphasis, filename, footnote, footnoteref, inlinemediaobject, link, literal, option, othercredit, phrase, quote, replaceable, revhistory, systemitem, trademark, ulink, userinput, xref.

title, titleabbrev.

For examples, see bibliography, section.
