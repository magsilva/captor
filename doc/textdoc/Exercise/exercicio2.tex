
Para utilizar a ferramenta, o engenheiro de aplicação deve possuir a especificação de requisitos de uma aplicação em um domínio que a ferramenta forneça apoio, editar os dados dessa especificação na ferramenta, gerar artefatos e utilizar esses artefatos no desenvolvimento de aplicações.

Para o exercício de utilização da ferramenta Captor são propostas as seguintes atividades:

\begin{itemize}
	\item Edição de uma especificação da ferramenta Captor que represente as tabelas e classes apresentadas nas Figuras \ref{fig:tabelasrelacionais} e \ref{fig:uml}.
	\item Geração dos artefatos.
	\item Teste dos artefatos gerados.
	\item Criação de zonas de segurança nos templates para adição de novos métodos nas classes geradas.
	\item Modificação dos artefatos gerados dentro das zonas de segurança.
\end{itemize}
